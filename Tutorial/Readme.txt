To setup the BOT fill in config.json:
token - your person DC token for hosting apps
prefix - for determining what is a command and what isnt
database-link - link to your remote mongoDB database
admin_ids - collection of DiscordUsers IDs who are supposed to have Admin privilages

-> Bot also requires specifis room names (given by Attributes on commands)
-> Bot also requires privileges (the simplest way is to just give him admin ones)
